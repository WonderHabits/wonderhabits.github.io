Github Blog(Jekyll)의 Build Command 는 다음과 같다.   

```bash
bundle exec jekyll serve
```

터미널에 해당 Command 를 입력하면 결과 화면을 볼 수 있는 링크가 나타난다.   
제대로 빌드 되었는지 확인 후 게시하면 된다.
